Features

Links

0.5.0 · Default: true

This features includes everything around links. With it enabled, you can...

  • ... select links
  • ... open links
features.links = true

Table of Contents

0.5.0 · Default: true

This feature enables the table of contents on the right side of the program. With it enabled you can also jump to a given header.

features.toc = true
